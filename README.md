# WeatherUIKitApp
WeatherUIKitApp - приложение для показа прогноза погоды по геопозиции пользователя

https://github.com/user-attachments/assets/f4162233-a2a5-4e00-a636-93a854448a25

## Описание приложения

- Одностраничное iOS-приложение на Swift с использованием UIKit с данными о погоде по геопозиции пользователя от сервиса weather API: https://www.weatherapi.com/
- Приложение состоит из одного экрана со списком погодных данных на ближайшие 3 дня
- Экран отображает погодную информацию: текущую, почасовую (показывает оставшиеся часы из текущего дня и все часы следующего), прогноз погоды на 3 дня
- Обработывается показ загрузки и ошибка, если что-то пошло не так, с кнопкой повторного запроса
- Дизайн на умотрение автора, можно ориентироваться на погодное приложение Apple

## Функциональные требования

- Запрашивать разрешение на использование геопозиции пользователя в самом начале (при отказе – использовать Москву)
- Данные получать из запросов API:

Для получения прогноза погоды нужно использовать следующий запрос:

http://api.weatherapi.com/v1/current.json?key=YOUR_API_KEY&q=LAT,LON

Пример использования запроса для получения прогноза на 3 дня:

https://api.weatherapi.com/v1/forecast.json?key=YOUR_API_KEY&q=LAT,LON&days=3
